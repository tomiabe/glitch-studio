# Glitch Studio

Interactive glitch art generator — upload any image and apply dithering, pixel manipulation, and colour-map effects in real time, directly in the browser.

**Features**

- Real-time glitch effects with live canvas preview
- Dither modes, colour-map presets, and adjustable glitch intensity
- Sidebar controls for fine-tuning every parameter
- Preset starter images so you can explore without uploading

## Run Locally

**Prerequisites:** Node.js 18+, a [Gemini API key](https://aistudio.google.com) (free)

```bash
npm install
# copy .env.example → .env.local and add your GEMINI_API_KEY
npm run dev
```
